# Project Structure

- `springai-ui/` Angular 21 application.
- `springai-chat-service/` Spring Boot Java 25 backend.
- `deployment/docker/` Docker Compose and local container assets.
- `deployment/k8s/base/` base Kubernetes manifests.
- `deployment/k8s/overlays/` environment overlays.
- `docs/adr/` architecture decision records.
- `docs/architecture/` overview and deployment documentation.
